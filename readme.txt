Pasos para jugar a Blackjack en Linux.

1. Entra desde la interfaz gráfica a la carpeta "Blackjack".
2. Haz clic con el botón secundario del ratón en cualquier lugar en blanco dentro de la vetana.
3. En el menú de opciones haz clic en "Abrir en una terminal". 
4. Desde la terminal escribe el siguiente comando: ./inicio.sh
5. La ventana de la terminal se te ajustará al tamaño del juego automáticamente.
6. Disfruta.

Para una mejor experiencia, se recomienda instalar el paquete xterm 
para redimensionar la ventana del terminal. 
Comando "sudo apt install xterm"
